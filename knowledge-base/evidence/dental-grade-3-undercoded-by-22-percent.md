# Dental grade 3 undercoded by 22%

Pre-v2-protocol chart audit of 320 dental cases across 18 clinics found 22% of true Grade 3 cases (moderate periodontitis with extractions) were billed as Grade 2. Methodology: independent secondary review of charts by Director of Clinical Quality + an external veterinary dental specialist; "true grade" set per the v2 grading rubric retroactively. Gap was concentrated in legacy-PIMS clinics (Avimark + eVetPractice) where coding workflows were less structured. Revenue impact: ~$650-1,400 per affected case; estimated $2.1-4.6M annual undercoding across the network. This finding drove the explicit dental-coding language in [[2026-02-18-clinical-protocol-v2-publish]].
